1) Create a MongoDB database named as "CarBooking"
2) Create 4 collections namely: cb_bookings, cb_cars, cb_reviews, cb_users
3) Import all 4 JSON files given in the ./collections folder
4) Change the connection string in .env file with your database details according to following: "DBURI=mongodb://127.0.0.1:27017/CarBooking"
5) Open terminal in "api" folder and run the below commands:
npm i
npm start
6) Open terminal in "Frontend" folder and run the below commands:
npm i
npm start
